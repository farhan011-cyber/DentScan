About This Tool

This web application uses state-of-the-art deep learning models to analyze dental X-ray images and detect the presence of caries. It helps in early detection and better dental health management.

Disclaimer

This web application is for demonstration purposes only. It is not intended for diagnostic or medical use. Always consult a qualified healthcare professional for dental health concerns.

Data

The images uploaded to this application are processed solely for the purpose of cavity detection. No personal or identifiable information is stored or used for any other purpose.
